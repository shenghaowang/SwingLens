# 🔭 SwingLens

A swing trading technical analysis web app built with React + Vite.

## Features
- 📈 5-year daily candlestick chart (Yahoo Finance data)
- 📊 MACD, RSI indicators
- 📉 SMA 50 & SMA 200 overlays
- 🎯 Automatic BUY/SELL signal detection
- 🚀 Auto-deployed to GitHub Pages

## Live App
https://shenghaowang.github.io/SwingLens/
